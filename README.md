# my_projects
My collection of AI and machine learning projects.

●画像認識AI（idol_classifier）
プロジェクト概要

2グループのアイドル写真を識別する画像認識AIを開発しました。
画像分類の基礎を実践的に学ぶことを目的に、Amazon SageMaker 上で環境を構築し、Pythonを用いて開発しました。モデルには TensorFlow/Keras を活用し、画像前処理には OpenCV を使用しています。

開発の目的

AI開発を学び始めた当初、実際に手を動かしながら「データ収集 → 前処理 → モデル構築 → 学習 → 推論」という一連の流れを経験したいと考えました。特に、画像を扱う課題に取り組むことで機械学習の具体的な仕組みを理解することを狙いました。

学んだこと

画像データの前処理や正規化といった基礎的なデータ加工手法

畳み込みニューラルネットワーク（CNN）の構造や学習プロセス

AWS上での開発環境構築とクラウドを用いた機械学習の実行方法

成果

複数枚のテスト画像を対象に正確にグループを識別するモデルを構築でき、画像認識AIの仕組みを理解することに成功しました。

●Mika_AI

英語で話しかけると、キャラクターが感情に応じて表情を変えながら返答してくれるAIを開発しました。
英語学習を「楽しく続けられる体験」に変えることを目指したプロジェクトです。

プロジェクト概要

音声認識でユーザーの発話を取得

ChatGPT APIで自然な英語の返答を生成

AWSのAmazon Pollyで返答を音声に変換

Flutterでキャラクターの表情を返答に合わせて切り替え

これにより、「英語で会話をするとキャラクターが笑顔になったり、照れたり、怒ったりしながら答えてくれる」インタラクティブな学習体験を実現しました。

開発の目的

従来の英語学習は単調で続けにくいという課題がありました。
そこで「キャラクターとの会話を通して学ぶ」という仕組みを作ることで、学習に楽しさと感情的なつながりを加えることを目指しました。

学んだこと

フロントエンドとバックエンドを組み合わせたサービス構築の流れ

APIキーや環境変数を用いたセキュアな開発手法

生成AIや音声技術を、実際のユーザー体験に落とし込む実践力

今後の展望

学習支援だけでなく、面接練習やカスタムキャラクターによる対話アプリなど、幅広い分野に応用できると考えています。
